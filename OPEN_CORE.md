# TramiFlow CRM - Open Core & PRO Features

TramiFlow follows an Open Core model. The core CRM and essential tools are open-source (Community Edition), while advanced features for scaling businesses are part of the PRO tier.

## 🟢 Community Edition (Open Core)
Available for free in this repository:
- **Gestión de Clientes**: Client profiles, history, and document management.
- **Kanban de Trámites**: Dynamic and configurable states for workflow tracking.
- **Template Builder (Basic)**: Drag & drop interface for workflow creation.
- **PDF Kit**: Full suite of 6 client-side tools (OCR, compression, etc.) with no server uploads.
- **Smart Documents (Basic)**: Essential document handling.
- **Authentication**: Solid multi-tenant auth architecture.

## 🔒 PRO Edition
Advanced infrastructure and features reserved for the commercial SaaS version:
- **Growth & Analytics**: Leads management, reporting, and WhatsApp integration.
- **Smart Limits**: Advanced Hard/Soft limit management with grace allowance.
- **Advanced Smart Documents**: Document re-optimization and advanced quality detection.
- **Command Vault (Admin)**: Super admin panel for multi-org management.
- **Lead Magnets**: Integrated landing pages for lead generation.
